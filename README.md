# Visual Database Schema Designer

A web app for visually designing database schemas: drag tables onto a canvas,
edit columns, draw foreign keys, and get live-generated PostgreSQL DDL.

## Status
Work in progress — scaffolding phase.

## Stack
- Frontend: React (Vite), React Flow, Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL (plain `pg` driver)
- Auth: JWT (email/password, bcrypt)
